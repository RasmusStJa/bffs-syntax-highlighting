# BEFORE Syntax Highlighting for VSCode
A basic syntax highlighter for our custom language

# Building
1. Install npm
2. Run `npm install vsce`
3. Run `vsce package`

# Installation
1. Download the most recent build from [tags](https://github.com/RasmusStJa/bffs-syntax-highlighting/tags)
3. Open VSCode and go to the Extensions tab
4. Click on the three dots on the top right
5. Click `Installl from VSIX`
6. Locate the .vsix file

# BFFS Syntax Highlighting for VSCode
A basic syntax highlighter for our custom language

# Building
1. Install npm
2. Run `npm install vsce`
3. Run `vsce package`

# Installation
1. Locate the .vsix file
2. Open VSCode and go to the Extensions tab
3. Click on the three dots on the top right
4. Click `Installl from VSIX`
